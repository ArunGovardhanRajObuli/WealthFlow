# Real Estate Service

> [!info] Property Taxation
> The **realEstateService.js** manages illiquid physical assets and dynamically calculates complex capital gains taxes using the Cost Inflation Index (CII).

## Core Responsibilities
1. **Indexation Engine**: Fetches the official Indian Cost Inflation Index (CII) either via a hardcoded fallback or by dynamically calling an external API (`https://raw.githubusercontent.com/arung/wealthflow-cii/main/cii.json`). It applies this indexation ratio (Sell Year CII / Buy Year CII) to the base cost to compute the `indexedCostBasis`.
2. **Capital Gains Splitting**: Calculates STCG (Short Term, < 2 years) or LTCG (Long Term, >= 2 years) depending on the holding period. Tax liabilities are distributed accurately based on the exact property ownership splits between family members.
3. **Cash Flow Integration**: Cross-references `ledger_lines` to calculate both Current Financial Year (FY) rent/expenses and Lifetime rent/expenses generated by the specific property.

# Reminders Routes

> [!info] Notifications Router
> The **reminders.js** route manages scheduled financial alerts.

## Endpoints
1. **GET `/`**: Retrieves all reminders.
2. **POST `/`**: Creates a new reminder.
3. **PUT `/:id`**: Updates an existing reminder.
4. **DELETE `/:id`**: Removes a reminder.

# ProtectionHub

> [!info] Shield Container
> The **ProtectionHub** is the structural root for all risk management, estate planning, and document storage features.

## Composition
It renders a tabbed interface containing:
1. `Insurance`
2. `HLVCalculator`
3. `FamilyEstate`
4. `SuccessionPlanner`
5. `DocumentVault`

It supports full keyboard accessibility (Arrow Keys, Home, End) to navigate between tabs efficiently without relying on React Router.
